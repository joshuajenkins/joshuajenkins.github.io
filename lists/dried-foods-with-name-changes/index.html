List of foods that change names once they are dried

Single-ingredient foods that change names via dehydration. Items cured using salt are excluded. Must have been exposed to in real life.

  • grape → raisin
  • jalapeño pepper → chipotle (smoke-dried)
  • poblano pepper → (chile) ancho
  • plum → prune
  • tomatoes → sun-dried tomatoes
  • wolfberry → goji berry
